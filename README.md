### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/wardvisual/engine-warvil.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. Create a copy of `.env.example` file

   ```sh
   cp .env.example .env
   ```

4. Enter your OPENAI API Key in `.env` file
   ```sh
   OPENAI_API_KEY = 'ENTER YOUR OPENAI API KEY'
   ```
